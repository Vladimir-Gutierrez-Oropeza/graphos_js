ALGORITMOS

Análisis de Algoritmos

Los algoritmos son los procedimientos que se construyen para la resolución de cualquier problema. De este modo, cuando se refiere a la construcción de un programa, nos estamos refiriéndo a la construcción de un algoritmo. El algoritmo no es un concepto proveniente del campo de la computación, sino que es un término matemático.

ALGORIMOS

GRAPHOS

Un grafo se define como un conjunto de nodos o vértices y aristas, donde cada arista relaciona a un par de nodos pertenecientes a un vértice.

Un algoritmo puede ser caracterizado por una función

Se dice entonces que un algoritmo calcula una función f. Entonces la entrada es una variable independiente básica en relación a la que se producen las salidas del algoritmo, y también los análisis de tiempo y espacio.

NOTAS

Cuando se tiene un problema para el cual debemos especificar un algoritmo solución, tendremos en cuenta varios puntos:

Se conoce un metodo de solución

debemos hacer un análisis del mismo para llegar a una solución, después de evaluar alternativas, exigencias y excepciones.

No se conoce un metodo de solución

se debe especificar exacta y completamente el método o procedimiento de solución en un lenguaje que se pueda interpretar fácilmente.

Solución de problemas
